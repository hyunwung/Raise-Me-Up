# Raise Me Up

### 만든이
 - FE : 서현웅
 - BE : 황하림

### 목적
- 개인의 목표를 달성하기위해 다른 유저들과 공유를 통한 동기 부여와 개인에 목표를 시각적으로 잘 알수있게 관리하기 위함
- 책을 만드는 듯한 자기 관리 웹 앱 만들기

### 주요 기능
- TO DO LIST를 남들과 공유하기(SNS…EX> 인스타그램, 페이스북,).
- 일 월 단위로 성취도 그래프 & 책
- 소셜 로그인을 사용(구글)
- todo mate를 참고하여 만들 예정임
